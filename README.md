# Cells::Slim

Support for Slim templates with Cells.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cells-slim'
```

## Usage

Outside of Rails, you need to include `Cell::Slim` in your cell.

```ruby
class CommentCell < Cell::ViewModel
  include Cell::Slim
```
